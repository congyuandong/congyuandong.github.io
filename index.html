-bash: /bin/rm: Argument list too long

发表于 2015-05-04   |   分类于 linux   |  

当目录下文件太多时使用rm命令会报如下的错误:

-bash: /bin/rm: Argument list too long

提示文件数目太多。解决的办法是使用如下命令:

ls | xargs -n 10 rm -fr ls

输出所有的文件名(用空格分割) xargs就是将ls的输出，每10个为一组(以空格为分隔符)，作为rm -rf的参数也就是说将所有文件名10个为一组，由rm -rf删除

Jquery each如何跳出循环并获得返回值

发表于 2015-03-05   |   分类于 WEB Dev   |  

看上去很简单的问题，怎奈我只有一颗后端的心，记录下好了

错误的写法

function checkRadio(){
    $(".reason .btn").each(function(){
        if($(this).hasClass('active')){
            return true;
        }
    });
    return false;
}

看上去很完美，但该方法只会返回false，原因是执行return true的时候并不会跳出each循环，因为：

  • return false：将停止循环 (就像在普通的循环中使用 ‘break’)；
  • return true：跳至下一个循环(就像在普通的循环中使用’continue’)。

正确的写法

综上，修改后的写法为:

function checkRadio(){
    var success = false;
    $(".reason .btn").each(function(){
        if($(this).hasClass('active')){
            success = true;
            return false;
        }
    });
    return success;
}
