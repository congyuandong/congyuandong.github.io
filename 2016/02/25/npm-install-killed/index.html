npm install killed

在服务器上部署node项目时使用npm install命令提示killed(已杀死)的，但是查看node_modules文件夹依赖创建了文件夹，所以以为是没有科学上网的缘故，使用淘宝镜像代替默认npm

结果还是提示killed，google之，得NPM gets killed no matter what，解决方法见这里，简言之就是内存不够。free -m查看服务器内存，果然消耗完了，而且阿里云的服务器默认swap空间为0
