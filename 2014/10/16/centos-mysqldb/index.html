Centos和MAC安装MysqlDB

发表于 2014-10-16   |   分类于 database   |  

分别在mac和centos上安装了无数次MysqlDB，还是记录一下吧，推荐大家先从简单的命令来，要不直接下代码安装巴拉巴拉的有时候就浪费时间了

最简单的

  • yum install MySQL-python
  • pip install MySQL-python

依赖不能忘！

  • yum install python-devel mysql-devel zlib-devel openssl-devel

因为一开始用mac装废了半天劲，所以在centos下我想都没想就直接下载源码去了，其实直接YUM就行，不过我特意试了第二种，会提示gcc错误，所以如果真的和我一样背，就乖乖下源码好了

暴力的

  • Download Source
  • tar -zxvf xxx.tar.gz
  • python setup.py install

提示mysql_config错误，这里只要修改site.cfg中mysql_config的路径即可

  • 修改为 /usr/bin/mysql_config

  • python setup.py install 即使尝试build错误也没有关系
