配置Django项目

选用uWSGI性能高而且内容占用低（和mod_wsgi比）

安装uWSGI

  • 可以看官方文档（别看那些乱七八糟的技术博客了，当然也包括这篇。。。）

  • pip install uwsgi

  • 因为有较多的django项目，为了让多个项目共用一个uwsgi（让它自己去做负载均衡，多nice～），需要安装virtualenv，pip即可

  • 需要在虚拟环境中安装软件，然后，你懂的。。。至少花了我一个小时各种pip

  • 在项目目录下新建uwsg配置文件

  • 启动方式uwsgi –ini uwsgi.ini
